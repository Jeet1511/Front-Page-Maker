## How to Use

### Step 1: Upload Institution Logo (Optional)
- Click "Choose Image" button
- Select your institution's logo (PNG, JPG, etc.)
- Preview will show the uploaded image

### Step 2: Fill in Your Details
- **Course/Subject Title**: e.g., "DSCC 3", "Final Year Project"
- **Student Name**: Your full name
- **Department**: e.g., "Computer Science", "Political Science Honours"
- **College/School Name**: Your institution name
- **University Name**: Your university (if applicable)
- **Roll Numbers**: College Roll No, University Roll No, Registration No

### Step 3: Add Custom Fields (Optional)
- Click "+ Add Field" button
- Enter custom field label and value
- Examples: "Session: 2023-24", "Semester: 6"
- Click the X button to remove unwanted fields

### Step 4: Choose a Template
- Click on any of the 4 template thumbnails on the right
- The preview will update immediately
- Try different templates to find the perfect design

### Step 5: Export Your Frontpage
Choose your preferred export format:

- **PDF** - High-quality A4 size PDF (recommended for printing)
- **PowerPoint** - Editable .pptx presentation
- **Word** - Editable .docx document
- **Image** - High-resolution PNG image

## Troubleshooting

### Preview not showing?
- Make sure JavaScript is enabled in your browser
- Refresh the page (F5 or Ctrl + R)
- Try opening in a different browser

### Export not working?
- Check that you're using a modern browser (Chrome, Edge, or Firefox)
- Ensure pop-ups are not blocked
- Wait for the preview to fully load before exporting
- Try exporting again if it fails the first time

### Logo not displaying?
- Make sure the image file is a valid format (JPG, PNG, GIF)
- Try a smaller image file (under 5MB)
- Check that the image preview shows before exporting

### PDF/Export quality issues?
- Wait a few seconds for the preview to render completely
- For PDF: Use the PDF export option for best quality
- For images: Ensure your browser window is not zoomed
